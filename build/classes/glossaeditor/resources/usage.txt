
Τρόπος χρήσης:
    java -jar slang.jar παράμετροι τοποθεσία_αρχείου_πηγαίου_κώδικα
π.χ.
    java -jar slang.jar /home/user/Documents/Hello_World.gls

Το αρχείο πηγαίου κώδικα μπορεί να παραληφθεί αν δε δοθεί η παράμετρος -c

Παράμετροι
    Όλες οι παράμετροι είναι προαιρετικές.

    -h  : εμφανίζει αυτό το μήνυμα

    -V  : εμφανίζει την έκδοση της εφαρμογής

    -c  : εκτέλεση διερμηνευτή μέσω γραμμής εντολών

            Με την παράμεμετρο αυτή πρέπει υποχρεωτικά να
            προσδιοριστεί αρχείο πηγαίου κώδικα
            Με την παράμετρο αυτή μπορούν να
            χρησιμοποιηθούν και οι παρακάτω:

            -i  : εκτέλεση βήμα-προς-βήμα

            -f τοποθεσία_αρχείου_εισόδου
                : Η είσοδος των εντολών ΔΙΑΒΑΣΕ
                  γίνεται από το αρχείο εισόδου,
                  διαβάζοντας μία γραμμή ανά εντολή ΔΙΑΒΑΣΕ
